# 🎨 Koto 图标替换指南

## 📋 快速操作步骤

### 方法一：使用快速替换工具（推荐）⭐

1. **保存图标图片**
   - 在聊天窗口中找到你想要的图标
   - 右键点击图片 → 选择"另存为"
   - 保存到桌面（任意文件名，如 `new_icon.png`）

2. **运行替换工具**
   - 双击打开 `快速替换图标.bat`
   - 将保存的图片文件拖拽到窗口
   - 按回车键

3. **完成！**
   - 工具会自动替换图标并重启应用
   - 查看任务栏和窗口标题栏的新图标

---

### 方法二：手动替换

1. 保存你的图标图片
2. 重命名为 `koto_icon.png`
3. 复制到 `Koto\assets\` 文件夹
4. 替换原文件
5. 重启 Koto.exe

---

### 方法三：命令行

```bash
# 保存图片后运行
python scripts\replace_icon.py "你的图片路径"
```

## 🖼️ 支持的图片格式

- ✅ PNG（推荐 - 支持透明背景）
- ✅ JPG/JPEG
- ✅ ICO
- ✅ BMP
- ✅ WEBP
- ✅ 其他常见格式

## 📐 图标要求

- **推荐尺寸**：256x256 或更大（正方形）
- **背景**：透明背景最佳
- **格式**：PNG格式效果最好

## 💡 实用提示

### 任务栏图标没变？

1. 完全关闭 Koto
2. 右键任务栏 Koto 图标 → 从任务栏取消固定
3. 删除 `%APPDATA%\Microsoft\Windows\Recent\AutomaticDestinations\` 中的缓存
4. 重新启动 Koto 并固定到任务栏

### 图标模糊？

- 使用更高分辨率的原图（512x512 或 1024x1024）
- 确保原图是PNG格式且清晰

### 想恢复原图标？

```bash
# 备份文件位于 assets 文件夹
# 文件名：koto_icon.png.backup 和 koto_icon.ico.backup
```
