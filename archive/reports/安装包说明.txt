🔧 手动启动方法
═════════════════════════════════════════════════════════════

方法 1: 命令提示符
  1. 按 Win + R
  2. 输入 cmd
  3. cd 到应用文件夹
  4. 输入: python koto_app.py

方法 2: PowerShell
  1. 在文件夹中右键
  2. 选择 "在此处打开 PowerShell 窗口"
  3. 输入: .\\run.bat

方法 3: 创建快捷方式
  1. 右键点击 run.bat
  2. 选择 "创建快捷方式"
  3. 将快捷方式移到桌面或开始菜单

🆘 遇到问题？
═════════════════════════════════════════════════════════════

问题: Python 未找到
解决: 
  1. 去 https://www.python.org/downloads/ 下载 Python
  2. 安装时 ⚠️ 勾选 "Add Python to PATH" ✅
  3. 重启电脑后再试

问题: 依赖安装失败
解决:
  1. 检查网络连接
  2. 手动运行: pip install -r requirements.txt
  3. 如果特定包失败，可能需要 Visual C++ 构建工具

问题: 应用无法启动
解决:
  1. 查看命令行窗口中的错误信息
  2. 检查 logs 文件夹中的日志文件
  3. 尝试更新 Python 或重新安装依赖
