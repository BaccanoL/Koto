## ❓ 常见问题

### Q: 应用启动失败？
```
A: 检查以下几点:
   1. 是否有 Python 3.8+ 
   2. 查看 logs/desktop_*.log 中的错误
   3. 尝试仅双击 run_desktop.bat
```

### Q: Agent 功能不工作？
```
A: 检查 API 密钥:
   1. 编辑: config/gemini_config.env
   2. 确保 GEMINI_API_KEY 有效
   3. 重启应用
```

### Q: 首次启动很慢？
```
A: 这是正常的！
   - 首次启动: 10-30 秒
   - 后续启动: 2-5 秒
```

## 🔧 配置

### 修改 API 密钥

编辑文件: `Koto\config\gemini_config.env`

```env
GEMINI_API_KEY=你的_API_密钥
MODEL=gemini-1.5-pro
```

### 修改用户偏好

编辑文件: `Koto\config\user_settings.json`

```json
{
  "theme": "dark",
  "auto_save": true
}
```

## 🐛 故障排查

### 问题：启动失败
```
解决:
□ 检查 Python (python --version)
□ 查看日志 (logs/desktop_*.log)
□ 运行: pip install -r requirements.txt
□ 重新启动
```

### 问题：Agent 不工作
```
解决:
□ 检查网络连接
□ 查看 API 密钥配置
□ 查看日志中的具体错误
□ 重启应用
```

### 问题：显示乱码
```
解决:
□ 检查系统编码 (应为 UTF-8)
□ 修改 config/user_settings.json 中的 language
□ 重启应用
```
