# Koto 图标替换说明

## 📖 使用方法

### 方法一：拖放替换（推荐）
1. 找到你要使用的图标文件（通常在下载文件夹中）
2. 将图标文件拖拽到 **`替换图标.bat`** 文件上
3. 松开鼠标，等待自动完成
4. 重启Koto应用查看新图标

### 方法二：手动替换
1. 找到你的新图标文件
2. 重命名为 `koto_icon.png` 
3. 复制到 `Koto/assets/` 文件夹，替换原文件
4. 重启Koto应用

### 方法三：命令行
打开PowerShell，运行：
```powershell
python replace_icon.py "你的图标文件路径"
```

## 📝 支持的格式
- PNG（推荐）
- JPG/JPEG
- ICO
- SVG
- BMP
- 其他PIL支持的格式

## 💡 提示
- 图标会自动调整为256x256像素
- 会自动保留透明背景（如果有）
- 原图标会自动备份为 `.backup` 文件
- 同时生成PNG和ICO两种格式

## 🔧 技术细节
- PNG用于系统托盘图标
- ICO用于窗口标题栏和任务栏
- 生成多尺寸ICO（16x16至256x256）
